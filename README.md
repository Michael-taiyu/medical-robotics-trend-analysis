# Medical Robotics Trend Analysis

## 📌 Project Overview
This project analyzes medical robotics trends using web scraping and data analysis.

## 📊 Data Sources
- Fierce Healthcare (news)
- PubMed (optional)

## ⚙️ Tech Stack
- Python
- Scrapy
- Pandas
- FastAPI (planned)

## 🚀 Goal
- Analyze trends in medical robotics (2015–2025)
- Build a resolver system for medical articles
